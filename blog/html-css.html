Inline & Inline-Block Display
A display property is what defines the display behaviour of an element, relative to other things around it. By default, a lot of elements are displayed as block, meaning that if we place them mid sentence, they would appear on a new line. Anything that then follows said element would appear on another new line. A few examples of default display block elements are

paragraph

division
and

H1

to

H6

heading elements.

Using inline elements allows us to keep a nicer flow within a webpage. They do this by keeping whatever element we are characterising on the same line as anything that precedes or follows it. There are also some elements that default to display inline. These include anchor, bold, and italics. When inline is set as the display property, the element will not be affected by any height or width properties. When inline-block is used, it gives us an element that we can change the size of using height and width properties. This can be useful for things like where you may want to make them larger to help with emphasis.
